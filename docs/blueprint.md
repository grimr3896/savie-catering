# **App Name**: CaterEase

## Core Features:

- Service Display: Showcase catering services with descriptions and pricing.
- Event Gallery: Display images of past events and catering setups.
- Booking Form: Enable clients to book catering services through a detailed form.
- Contact Form: Allow users to contact the catering company for inquiries and custom requests.
- AI-Powered Menu Suggestion Tool: Provide personalized menu suggestions based on event type, dietary preferences, and budget, with AI reasoning about optimal food choices.
- Event Management: Manage booking events via Cloud SQL.

## Style Guidelines:

- Primary color: Sophisticated gold (#B8860B) to evoke luxury and quality.
- Background color: Soft beige (#F5F5DC) for a clean, elegant backdrop.
- Accent color: Deep brown (#8B4513) to add contrast and warmth.
- Headline font: 'Playfair', a modern serif with an elegant, fashionable feel.
- Body font: 'PT Sans', a humanist sans-serif with a modern feel. For body text.
- Use elegant, minimalist icons representing different catering services and event types.
- Ensure a clean and structured layout, with clear sections for services, gallery, and booking.
- Subtle transitions and animations on image hover and page transitions to enhance user engagement.
